# Sample Dataset
This `sample` folder contains data from a real run of the tool and subsequent testing using a Hashtopolis custer for cracking.
- [password_analysis_xlsx.zip](password_analysis_xlsx.zip) contains an Excel xlsx spreadsheet analysing the dataset and results of cracking and checking HIBP
- [credentials.csv](credentials.csv) contains the raw username and password output created by the model using personas
- There are files containing password dumps/hashlists for these credentials in various formats for testing
